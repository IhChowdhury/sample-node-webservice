<p align="center">Sample res Webservice with NestJs Framework</p>


## Description
A Sample webservice for rest api.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

http://localhost:3000/api-doc
